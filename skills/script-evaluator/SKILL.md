# Script Evaluator — Write & Score Viral Scripts

> Evaluate any script for viral potential, or write one from scratch. Same virality brain used in AI clipping, applied BEFORE filming.

## Two Modes

| Mode | When to Use |
|------|------------|
| **Evaluate** | User has a script → score it, find weaknesses, suggest improvements |
| **Write** | User has a topic/idea → generate a viral-ready script from scratch |

---

## Mode 1: Script Evaluation

### Step 1 — Score the Script (0-100)

Apply these 8 criteria to the full script. Each contributes to a composite score:

| # | Signal | Weight | What to Check |
|---|--------|--------|---------------|
| 1 | **Hook Strength** | 25% | Do the first 1-2 sentences create immediate curiosity? Would someone stop scrolling? |
| 2 | **Emotional Resonance** | 20% | Does the script trigger a feeling? Surprise, anger, laughter, inspiration? |
| 3 | **Opinion Strength** | 15% | Does it take a clear stance? Is there something to agree/disagree with? |
| 4 | **Revelation Value** | 12% | Does the viewer learn something surprising? Is there a "wait, really?" moment? |
| 5 | **Tension/Conflict** | 8% | Is there a problem, challenge, or opposing viewpoint that creates tension? |
| 6 | **Quotability** | 7% | Is there at least one line someone would screenshot and share? |
| 7 | **Story Arc** | 7% | Does it have a clear beginning → build → payoff structure? |
| 8 | **Practical Takeaway** | 6% | Does the viewer leave with something actionable? |

### Step 2 — Identify Weaknesses

For each signal scoring below 60%, flag it with a specific fix:

```markdown
## Script Evaluation: "Why AI Won't Replace Developers"

**Overall Score: 62/100** — Decent content, weak hook, missing emotional peaks

### Signal Breakdown
| Signal | Score | Assessment |
|--------|-------|-----------|
| Hook | 35/100 | ❌ Opens with "Today I want to talk about..." — generic, no curiosity |
| Emotional | 55/100 | ⚠️ Gets passionate at 0:45 but starts flat |
| Opinion | 80/100 | ✅ Strong contrarian take — "AI makes bad devs worse, good devs better" |
| Revelation | 70/100 | ✅ Good stat at 1:15 about AI-assisted code quality |
| Tension | 40/100 | ❌ Never addresses the counter-argument seriously |
| Quotability | 75/100 | ✅ "AI is a lever, not a replacement" — shareable |
| Story Arc | 50/100 | ⚠️ Builds well but ending is abrupt |
| Practical | 65/100 | ✅ Has 2 actionable tips but buried mid-script |

### Top 3 Fixes (highest impact)
1. **Rewrite the hook** (current: 35 → target: 80+)
   - Before: "Today I want to talk about AI and software development."
   - After: "Everyone says AI will replace developers. They're wrong — and here's the proof."

2. **Add tension at 0:30** (current: 40 → target: 70+)
   - Insert: "And before you comment 'cope' — let me show you the data."

3. **Restructure ending** (current: 50 → target: 75+)
   - Move the practical tips to the end as a clear CTA
   - End with the quotable line, not after it
```

### Step 3 — Provide Rewrite Options

For the weakest sections, offer 2-3 alternative versions ranked by viral potential.

## Mode 2: Script Writing

### Input Required

Ask the user for:
1. **Topic/idea** — What's the video about?
2. **Target platform** — TikTok/Reels (30-90s) vs YouTube Shorts (30-60s) vs YouTube (3-10 min)
3. **Content type** — Hot take, tutorial, story, reaction, listicle, behind-the-scenes
4. **Tone** — Professional, casual, edgy, inspirational, humorous
5. **Target audience** — Who should this resonate with?

### Script Structure Templates

#### Template A: Hook → Proof → CTA (30-60s)

```
HOOK (0-3s):
[Bold claim or curiosity gap — THE most important line in the entire script]

BRIDGE (3-8s):
[Quick context — why the viewer should care RIGHT NOW]

PROOF (8-45s):
[Evidence, examples, story — the substance that delivers on the hook's promise]
- Point 1 (strongest — frontload value)
- Point 2 (supporting)
- Point 3 (if time allows)

PAYOFF (45-55s):
[The "aha" moment or key takeaway — this is what gets shared]

CTA (55-60s):
[What should the viewer do? Follow, save, comment, try it]
```

#### Template B: Story → Lesson (45-90s)

```
HOOK (0-3s):
[Dramatic moment from the MIDDLE of the story — in medias res]
"I was sitting in the meeting when my boss said something that changed everything."

SETUP (3-15s):
[Quick backstory — just enough context]

BUILD (15-50s):
[Rising action — what happened, tension building]
[Include at least ONE moment of doubt/conflict]

CLIMAX (50-65s):
[The twist, reveal, or peak emotional moment]

LESSON (65-80s):
[What you learned — make it universal, not just personal]

CLOSE (80-90s):
[Memorable one-liner that captures the whole story]
```

#### Template C: Myth-Bust (30-60s)

```
HOOK (0-3s):
"Everyone thinks [common belief]. Here's why they're wrong."

MYTH (3-10s):
[State the myth clearly — the viewer should nod along]

EVIDENCE (10-40s):
[Present counter-evidence — data, experience, logic]
[Each point should feel like a mic drop]

REFRAME (40-50s):
[Offer the new mental model — "Instead of X, think of it as Y"]

CLOSE (50-60s):
[Quotable one-liner summarizing the new truth]
```

#### Template D: Listicle / Tips (45-90s)

```
HOOK (0-3s):
"X things I wish I knew about [topic]" / "Stop doing [bad thing], do [good thing] instead"

ITEMS (3-70s):
[Each item follows: setup (2s) → reveal (3-5s) → why it matters (2-3s)]
- #1: [Strongest/most surprising — always frontload]
- #2: [Most practical/actionable]
- #3: [Most emotionally resonant]
- #4-5: [If time allows — save the second-best for last]

CLOSE (70-85s):
[Recap the theme, not the items — "The point is..."]
[End with highest-impact item or callback to hook]
```

## Hook Library — Proven Patterns

### Curiosity Gaps
- "The [X] nobody talks about..."
- "I found out why [unexpected thing happens]..."
- "There's a reason [well-known person/company] does [surprising thing]..."
- "What happens when you [unlikely action]? I tried it."

### Contrarian Takes
- "Unpopular opinion: [bold claim]"
- "[Common advice] is actually making things worse."
- "I stopped [common practice] and here's what happened..."
- "Everyone's doing [X]. Here's why that's a mistake."

### Social Proof / Authority
- "After [N] years of [doing X], this is what I've learned..."
- "I [impressive credential/experience] — here's what nobody told me..."
- "[Impressive stat about the result]"

### Emotional Triggers
- "This completely changed how I think about [topic]..."
- "I was today years old when I learned..."
- "The moment I realized [vulnerable admission]..."

### Direct Address
- "If you [specific behavior], watch this."
- "This is for everyone who [relatable struggle]."
- "Stop scrolling if you [qualifier that makes viewer feel seen]."

## Platform-Specific Guidelines

### TikTok / Instagram Reels (30-90s)

```
- Hook MUST land in first 1.5 seconds (or they swipe)
- Speaking pace: 160-180 words per minute (fast but clear)
- Captions are MANDATORY (85% watch without sound)
- End with a loop-friendly moment (seamless restart = algorithm boost)
- Engagement prompt: "Comment [X] if you agree" / "Save this for later"
- Optimal length: 45-60s (completes in one watch, algorithm favors >50% watch-through)
```

### YouTube Shorts (30-60s)

```
- Hook in first 2 seconds
- Speaking pace: 150-170 wpm (slightly slower than TikTok)
- Value-dense — Shorts viewers are more info-seeking
- End with a channel pitch if trying to convert to long-form viewers
- Optimal length: 40-55s
```

### YouTube Long-form (3-10 min)

```
- Hook in first 5 seconds, then a "preview" of what's coming (0-15s)
- Speaking pace: 140-160 wpm
- Structure needs clear chapters/sections
- Include a "proof" moment in first 30s (why you're credible on this topic)
- Retention dips at 30s, 2min, 5min — place micro-hooks at these points
- End with a related video recommendation
```

## Evaluation Rubric Quick Reference

| Score Range | Verdict | Action |
|-------------|---------|--------|
| 90-100 | 🔥 Exceptional | Film immediately — this has breakout potential |
| 75-89 | ✅ Strong | Minor tweaks to hook/ending, ready to film |
| 60-74 | ⚠️ Decent | Needs work on 2-3 signals — rewrite weak sections |
| 40-59 | 🟡 Weak | Major restructuring needed — consider different angle |
| 0-39 | ❌ Not viable | Scrap and start fresh — core concept lacks viral potential |

## Script Improvement Workflow

```
1. USER provides script (or topic for writing)
2. EVALUATE — Score all 8 signals
3. IDENTIFY — Top 3 weaknesses with highest improvement potential
4. REWRITE — Provide 2-3 alternatives for each weak section
5. RE-EVALUATE — Score the improved version
6. ITERATE — Until score ≥75 or user is satisfied
```

### Improvement Priority Order (fix these first)

1. **Hook** — Highest impact, easiest to fix
2. **Story arc / structure** — Restructuring can lift everything
3. **Emotional peaks** — Add specific moments of feeling
4. **Quotable lines** — Craft 1-2 shareable sentences
5. **Tension** — Add counter-arguments or obstacles
6. **CTA / ending** — Clear, memorable close

## Integration with AI Clipping

When a script has already been filmed and you're extracting clips:

1. **Pre-film evaluation**: Score the script BEFORE filming → identify the 3-5 segments most likely to go viral as standalone clips
2. **Mark clip boundaries in the script**: Add `[CLIP START]` / `[CLIP END]` markers where natural standalone clips exist
3. **Post-film extraction**: Use the `ai-clipping` skill with the virality scores as priors — segments you pre-identified as viral in the script should get a scoring bonus when found in the transcript

This creates a **feedback loop**: script evaluation → filming → clip extraction → performance data → better script evaluation next time.
